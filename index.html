حالت اول: مرورگر آزاد (Web Proxy)

حالت دوم: اتصال کل گوشی و مودم

دریافت تنظیمات مخفی جهت اعمال در پروکسی وای‌فای گوشی یا مودم برای آزاد شدن برنامه‌هایی مثل تلگرام، واتساپ و سایفون.

🌐 مرورگر آزاد اینترنت

آدرس سایت مورد نظرتان را وارد کنید تا از طریق سرور امن باز شود.

⚙️ تنظیمات مخفی آزادکننده شبکه (سیستمی)

با وارد کردن مشخصات زیر در بخش تنظیمات پروکسی شبکه وای‌فای (Wi-Fi Proxy) گوشی یا مودم خود، ترافیک برنامه‌هایی مثل تلگرام، واتساپ یا ابزارهایی مثل سایفون را برای عبور از سد فیلترینگ آزاد کنید:

Proxy Type: HTTP / HTTPS
Server Address: ...
Port: 443
PAC URL (Auto): ...

💡 **راهنمای اندروید و آیفون:** وارد تنظیمات Wi-Fi گوشی شوید، روی علامت چرخ‌دنده یا علامت (i) شبکه متصل شده بزنید. بخش Proxy را روی حالت **Auto** گذاشته و آدرس PAC URL بالا را قرار دهید. (یا روی Manual گذاشته و Server و Port را دستی ست کنید).

🚀 **کمک به سایفون:** در تنظیمات سایفون به بخش Upstream Proxy بروید و هاست پروکسی را برابر با آدرس سرور بالا و پورت را روی 443 قرار دهید تا سرعت کانکت شدن آن چندبرابر شود.
