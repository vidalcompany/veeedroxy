باز کردن وب‌سایت‌ها و استریم ۱۰۰٪ فیلم و یوتیوب مستقیماً در مرورگر بدون نیاز به نصب برنامه.

حالت دوم: اتصال کل گوشی و مودم

دریافت تنظیمات مخفی جهت اعمال در پروکسی مودم یا گوشی برای آزاد شدن تلگرام، واتساپ و سایفون.

🌐 مرورگر آزاد اینترنت

آدرس سایت مورد نظرتان را وارد کنید تا از طریق سرور امن باز شود.

⚙️ تنظیمات مخفی آزادکننده شبکه (سیستمی)

با وارد کردن مشخصات زیر در بخش تنظیمات پروکسی شبکه وای‌فای گوشی یا تنظیمات مودم خود، ترافیک برنامه‌هایی مثل تلگرام، واتساپ یا ابزارهایی مثل سایفون را آزاد کنید:

Proxy Type: HTTP / HTTPS
Server Address: ...
Port: 443
PAC URL (Auto): ...

💡 **راهنمای موبایل:** وارد تنظیمات Wi-Fi شوید، روی نام شبکه متصل بزنید. بخش Proxy را روی حالت **Auto** گذاشته و لینک PAC URL بالا را قرار دهید (یا روی Manual گذاشته و Server و Port را دستی ست کنید).

🚀 **کمک به سایفون:** در تنظیمات سایفون به بخش Upstream Proxy بروید و هاست پروکسی را برابر با آدرس سرور بالا و پورت را روی 443 قرار دهید تا فوراً متصل شود.
